## 程法第6条

什么是自律？

自律并不是在每天都做同样的事情，而是每天都做正确的事情，这个事情一定是对自己或别人长期有好处的事，
首先要过自己这关，自己认为对的事，然后再是不触及法律和道德。
自律还有一层意思就是：克制自己做不正确的事！当还有未完成的学习任务，不能娱乐至死。
